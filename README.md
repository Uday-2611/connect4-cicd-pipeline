# Connect 4 CI/CD Pipeline

A multiplayer Connect 4 game with full CI/CD pipeline using Jenkins, Docker, Kafka, and PostgreSQL.

## Tech Stack
- Frontend: React.js
- Backend: Node.js + Express
- Database: PostgreSQL
- Messaging: Apache Kafka
- CI/CD: Jenkins
- Containerization: Docker

## Quick Start
```bash
docker-compose up --build
```

Visit http://localhost:3000
